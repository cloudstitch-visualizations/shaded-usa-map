Using the Widget:

Your project page on Cloudstitch has instructions for usage on particular content management systems. Here is an example of using the widget if you have access to raw HTML.

  1. Include the Cloudstitch Javascript in you page:

    <script src="http://static.cloudstitch.io/cloudstitch.js"></script>

  2. Include the following short code where you want your widget:

    <div widget="project-templates/electoral-map-visualization></div>
